Angular

overview
    • introduced by Google
    • open source
    • released in September 2016
    • current version is 4 (released in March 2017)
    • v3 was skipped to avoid semantic version issues
    • MVC

Angular

pros & cons
    • focus on components
    • TypeScript
    • event handling capabilities
    • Angular CLI
    • easy to scale
    • better performance
    • regular DOM
    • complex - may be difficult to learn
    • performance problems on mobile



ReactJS

overview
    • introduced by Facebook
    • open source
    • released in May 2013
    • current version is 15.6.1
    • V layer in the MVW models

ReactJS

pros & cons
    • focus on components
    • JSX
    • virtual DOM
    • high performance
    • light-weighted
    • redux & flux architecture
    • problems with other libraries
      DOM manipulation
    • no router



Vue.js

overview
    • introduced by Evan You
    • open source
    • released in February 2014
    • current version is 2.4.4
      (released in September 2017)

Vue.js

pros & cons
    • small size
    • virtual DOM
    • flexibility
    • simplicity in syntax
      & project setup
    • fast rendering
    • language barrier
    • closed community development

Aurelia

overview
    • introduced by EisenbergEffect, Durandal Inc.
    • open source
    • released in June 2016
    • current version is 1.1.4

Aurelia

pros & cons
    • clean & simple - plain JS
    • focused on Web Standards
    • good integration with others
    • easy to work with
    • ES5, ES6, TS,... support
    • commercial & enterprise support
    • small community
